# 3033.design

Hardware & robotics studio site. Built on Next.js 15 (App Router) + Tailwind.

## Dev

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000.

## Structure

- `app/` — pages (App Router)
- `components/` — shared UI (Chrome, ThemeToggle)
- `legacy/` — original static HTML preserved for reference
